1 · Setup — add your API keys

Keys are saved to the local .env file on the server and never leave your machine. Nothing works until both keys are set.

Where to get the keys:
• Google Gemini (free) — aistudio.google.com/apikey → “Create API key” (starts with AIza). Used for answering.
• Pinecone — app.pinecone.io → API Keys (free “Starter” tier is enough). The index is created automatically on first use.
Embeddings run locally (fastembed) — no key needed for those.
